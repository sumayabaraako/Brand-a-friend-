# Brand a Friend

## Projectomschrijving

Brand a Friend is een project waarbij een interactieve digitale ervaring wordt ontworpen voor een gekozen persoon. Voor dit project heb ik Sanne gekozen.

Door middel van interviews en onderzoek heb ik gekeken naar haar interesses, persoonlijkheid en voorkeuren. Deze informatie vormde de basis voor het ontwerp en de inhoud van de website.

## Concept

Uit het onderzoek bleek dat Sanne veel plezier haalt uit lezen. Daarom heb ik ervoor gekozen om de ervaring vorm te geven als een interactief digitaal boek.

In plaats van een standaard website ontdekt de gebruiker de inhoud door pagina's te openen en door het boek te bladeren. Op deze manier sluit de vorm van het project aan bij een interesse die belangrijk is voor Sanne.

## Doel

Het doel van dit project was om een persoonlijke en interactieve ervaring te ontwerpen die aansluit bij de identiteit van Sanne. Door onderzoek te combineren met vormgeving en ontwikkeling ontstaat een digitale ervaring die speciaal voor haar is gemaakt.

## Functionaliteiten

- Interactief boekconcept
- Digitale bladerervaring
- Visuele presentatie van persoonlijke informatie
- Responsief ontwerp
- Storytelling gebaseerd op onderzoeksresultaten

## Technologieën

- HTML5
- CSS3
- JavaScript

## Proces

Het project begon met interviews en onderzoek naar Sanne. De verzamelde informatie werd vervolgens gebruikt om een concept te ontwikkelen dat aansloot bij haar interesses.

Vanuit dit concept zijn de structuur, vormgeving en interactieve elementen ontworpen en uitgewerkt tot een werkende website.
